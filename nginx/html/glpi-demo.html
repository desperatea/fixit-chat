Как это работает

Эта страница имитирует страницу GLPI, на которой пользователь уже авторизован. Виджет чата внизу справа подключён с подписанным токеном — форма ввода имени/телефона пропущена.

Генерация токена...
Нажмите на кнопку чата внизу справа. Сессия создастся автоматически с данными из GLPI — без формы.
  1. GLPI знает пользователя (PHP $_SESSION)
  2. Сервер генерирует HMAC-SHA256 токен с данными пользователя
  3. Виджет получает токен через data-glpi-token
  4. Бэкенд проверяет подпись и создаёт сессию
